# X Agent Registry

## Enterprise AI Agent Discovery, Governance and Adoption Platform

### Version

1.0

### Purpose

X-Company has rapidly adopted AI tools such as GitHub Copilot, Atlassian Rovo and custom AI agents. Teams independently create agents, skills and prompts across repositories, resulting in limited visibility, duplicated efforts and governance challenges.

X Agent Registry provides a centralized platform to:

* Discover all agents and skills across repositories
* Maintain an enterprise-wide inventory
* Track usage and adoption
* Promote reuse across teams
* Identify dormant and duplicate agents
* Provide governance and leadership reporting

# Agent Definition Standard

Each repository contains agent definition files.

Example:

agent.yaml

agent.md

agent.json

Mandatory metadata:

agentId
agentName
description
ownerTeam
businessUnit
repository
version
status
skills
tags
visibility
telemetryEnabled
createdDate
lastModifiedDate

Example Frontmatter:

---

agentId: AGT-001
agentName: API Test Generator
description: Generates API test cases from OpenAPI specs
ownerTeam: QA COE
businessUnit: Engineering
version: 1.0.0
status: ACTIVE
skills:

* Swagger Parsing
* Test Generation
  tags:
* api
* automation
  telemetryEnabled: true
